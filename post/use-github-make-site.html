如何使用GitHub來架設一個網站

首先，你需要一個GitHub帳號（廢！！！！！話！！！！！）
登入帳號後，在首頁點這顆綠綠的按鈕

輸入你的儲存庫名稱
如果命名為[你的使用者名稱].github.io，那你的網站網址就會是[你的使用者名稱].github.io，
如果不是，那你的網站網址就會是[你的使用者名稱].github.io/[儲存庫名稱]

點擊Create repository

恭喜你，你已經建立好一個儲存庫了！
你應該會看到這些東西：

不要慌，接下來就點creating a new file來建立一個html檔，或是點uploading an existing file來上傳裝置裡的檔案

如果想讓網頁出現在首頁，請把它命名為index.html

點擊Commit new file或Commit changes來儲存你對網站的變更
